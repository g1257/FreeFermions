# Quick Start

## Dependencies and how to download:

<pre>
cd someDirectory
git clone https://github.com/g1257/PsimagLite
git clone https://github.com/g1257/FreeFermions
</pre>

## Compiling and Running

<pre>
cd FreeFermions/examples
perl make.pl
./cicj 8 4
</pre>

## To update the code:

<pre>
cd someDirectory
cd PsimagLite
git pull
cd ../FreeFermions
git pull
</pre>
